# SaifCTL (VS Code extension)

A VS Code extension to manage zero-trust, containerized AI coding swarms with the `saifctl` CLI.
Write your specs, and let the agents build the feature until it passes your hidden tests.

Install the CLI from npm:

```bash
npm install -g @safe-ai-factory/saifctl
```
