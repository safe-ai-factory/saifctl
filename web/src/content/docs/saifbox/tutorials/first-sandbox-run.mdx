# Tutorial: your first sandbox run

This tutorial takes you from a fresh install to a completed agent run in an isolated container. By the end you will have run an agent, inspected its output, and optionally applied its changes to your working tree.

## What you will need

- Docker Desktop (or Docker Engine) running on your machine
- `saifctl` installed — run `saifctl doctor` to verify
- A project directory with at least one file tracked by git
