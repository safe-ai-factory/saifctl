# Saifbox

**Saifbox** is the isolated sandbox layer of the Safe AI Factory stack. It runs any agent CLI — OpenClaw, Claude Code, or others — inside an ephemeral Docker container so the agent never touches your host filesystem, secrets, or credentials unless you explicitly opt in.

## Who this is for

- Developers who want to run AI coding agents without giving them unrestricted host access.
- Teams evaluating agent output before applying it to a real working tree.
